# Базовий React проєкт

Цей проєкт містить компонент кнопки та текстового поля, які приймають props для зручної взаємодії.

## Як запустити проєкт

1. Клонуйте репозиторій:
```bash
    git clone <URL>
```

2. Встановіть залежності:
 ```bash
    npm install
 ```
3. Запустіть проєкт:
 ```bash
    npm run dev
 ```
